UBBEY

The First Blockchain Personal Cloud

Ubbey box is a device designed by Universal Labs for family or personal users. Ubbey box is also a full node for the Ubbey network, which means it acts like the “miner” in Bitcoin network. With Ubbey box, a person can save everything in one place and access it from anywhere with blazing-fast performance. Users can also use Ubbey box to mine the YOU coin.
